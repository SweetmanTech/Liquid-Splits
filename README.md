### ThirdWeb Release

1. Visit [LiquidSplit](https://thirdweb.com/sweetman.eth/LiquidSplit) on Thirdweb.
1. Click "Deploy Now" button.
1. Select any network.
1. Deploy LiquidSplit.
1. Send native gas token (`ETH`/`MATIC`) to use your liquid split.
